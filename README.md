## ✨ 功能特性

* **双练习与生产模式切换**：
  * **🆓 免费模式（无需 Key）**：完全免注册，基于纯本地高容错渲染引擎。内置 12 种核心职场场景，输入关键表单字段即可一键拼接出高可用邮件草稿。
  * **🤖 AI 模式（需 API Key）**：解锁流式深度生成。支持复杂的中文意图直接转化、Chinglish 语法地道润色、以及读取完整上下文关联的复杂邮件串深度回复。
* **12 种主流职场场景全覆盖**：
  * **求职申请 (Job Application)**：支持动态应聘岗位、目标公司、核心优势（一句话）精准匹配。
  * **面试跟进 (Interview Follow-Up)**：支持面试日期转化、根据跟进目的（仅感谢、仅确认进度、感谢+确认进度）动态派生不同话术。
  * **商务合作 (Business Collaboration)**：支持细分合作方向、明确期望下一步（如安排 20 分钟视频通话）。
  * **日常办公全场景**：感谢信 (Thank You)、请假申请 (Leave Request)、会议邀请 (Meeting Invitation)、催促回复 (Follow-Up)、拒绝 Offer (Decline Offer)、投诉 (Complaint)、道歉 (Apology)、介绍引荐 (Introduction)、离职通知 (Resignation)。
* **多维度表达微调机制**：
  * **三档正式度自由切换**：**正式**（面对上级/客户/陌生人）、**标准**（职场通用流）、**轻松**（熟识同事，自动带入更具亲和力的社交语态）。
  * **📚 72 条高频外企句型库**：针对开头句、过渡句、结尾句划分，涵盖标准美式、地道英式（*Cheers!*）、活泼澳式（*Thanks heaps!*）等常用语体系，点击即可自动插入结果编辑区。
* **可视化中英对照与动态草稿编辑**：
  * 生成结果框支持**直接就地编辑修改**，文本框具备高度自适应（Auto-height）动态拉伸能力。
  * 桌面端支持左右分栏中英双语对照展示，移动端自适应一键平滑折叠切换。
* **🔒 隐私保护与全链路安全防护**：
  * **智能敏感词拦截**：对提交给大模型的数据进行本地正则表达式审查，自动识别潜在的**邮箱地址**、**手机号**、**金额信息**、**电话号码**并弹出高亮黄色警告。
  * **防并发与超时硬锁**：内置单次请求 60 秒硬超时熔断机制，并在生成期间禁用连点按钮，有效降低接口额度损耗。
* **无感存储与多维导出**：
  * 基于 `localStorage` 自动无感缓存最新 50 条生成历史，支持一键星标收藏夹置顶、一键复用载入或彻底销毁。
  * 支持一键单项/全部复制、直接唤起系统默认邮件客户端 (`mailto`) 发送，或一键导出为标准的 `.txt` 纯文本文件。

## 🔑 两种模式对比与配置

| 模式 | API Key 依赖 | 核心输入形式 | 专属核心功能 |
| :--- | :---: | :--- | :--- |
| **🆓 免费模式** | **无需** | 场景关联表单项 + 正式度分档 | 基于内置固定精选句型进行纯本地拼接，适合快速出常规函件 |
| **🤖 AI 模式** | **必填** | 自由文本描述 / 英文原稿 / 完整往来 Context | 解锁流式深度中文生成、修改原因逐句对照说明、支持 4 款顶级模型切换 |

### 🤖 接入模型说明
系统无缝对接 OpenAI 标准兼容接口，提供以下主流模型矩阵支持：
* `deepseek-chat`：**默认推荐**。国内顶尖大模型，商务英语表达极其自然，经济高效。
* `gpt-4o-mini`：流式响应极其敏捷，处理常规润色任务速度极快。
* `gpt-4o` / `claude-sonnet-4-6`：Premium 级别高阶模型，适合处理具有复杂上下文博弈的商务谈判邮件。
> 💡 系统接口默认通过 `https://jotarou.com/v1` API 网关转发至大模型服务商，网关不持久化任何对话内容。
