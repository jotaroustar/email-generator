# 英文邮件生成器
---

## 工具简介

专为中国职场人设计的英文邮件辅助工具。免费模式提供 12 种场景的地道英文邮件模板，AI 模式支持中文生成、Chinglish 润色和邮件串回复。

## 核心功能

**免费模式（无需注册）**
- 12 种职场场景：求职申请、面试跟进、会议邀请、请假申请、商务合作等
- 正式 / 标准 / 轻松三档语气
- 72 条常用句型库，按场景过滤，点击即用
- 一键复制 / 下载 .txt / 📧 唤起邮件 App

**AI 模式（需 API Key）**
- 📝 中文生成：用中文描述意图，直接输出地道英文邮件
- ✨ Chinglish 润色：粘贴草稿，逐句对照说明改了什么、为什么改
- ↩️ 回复邮件串：粘贴完整 Thread + 中文要点，AI 读取上下文生成回复
- 中英对照展示，桌面端左右分栏，移动端一键切换
- 输出框可直接编辑，修改后保存

## 使用方式

1. 访问 [jotarou.com/tools/email/](https://jotarou.com/tools/email/)
2. 免费模式：选场景 → 填信息 → 生成
3. AI 模式：填写内容 → 输入 API Key → 生成

> API Key 在 [jotarou.com](https://jotarou.com) 注册后获取，支持 DeepSeek / GPT-4o / Claude 等多个模型。

## 隐私说明

输入内容经 jotarou.com API 网关转发至大模型服务商，网关不持久化对话内容。  
建议将邮件中的真实公司名、人名、报价等替换为占位符后再提交。
